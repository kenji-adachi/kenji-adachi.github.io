Project: Pipesim

Senior Project

Pipesim is the senior design project for my university. Me and two other team members workrf with a client to complete the requirements of the client. We worked on updating and adding features to an existing software for the client to use in research. The premise of the project was working with Python to make a pipe water flow simulation. We held weekly and bi-weekly meetings with the client and held team meetings to cover any current updates to the project or requirements. Most of the information must remain disclosed but we were able to make the file input and output system easier and more intuitive. My teammates also made the software a working prototype into a website. The main challenges were looking at the previous code for the software to understand the inner workings to be able to identify any areas for improvement and suggestions to the client. In the end we presented our work to others with a poster during a walk around event and had met the needs of the client.
